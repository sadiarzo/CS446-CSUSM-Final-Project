# CS446-CSUSM-Final-Project
Building a scalable cloud-native architecture for multiplayer games using GCP and Kubernetes.

**Overview**
This project investigates the use of cloud-native infrastructure to improve scalability and reduce latency in multiplayer online gaming environments. The system is designed using Google Cloud Platform and Kubernetes to dynamically scale game server instances based on player demand.

**Objectives**
- Design a scalable multiplayer server architecture using GKE
- Implement containerized game server instances
- Evaluate system performance under simulated player load
- Compare cloud-based deployment with centralized server architecture

**Technologies**
Google Cloud Platform (GCP)
Google Kubernetes Engine (GKE)
Docker 
Cloud Pub/Sub 
Firestore / Spanner
